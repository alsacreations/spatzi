Type de contraste
WCAG 2.1 : Web Content Accessibility Guidelines . Standard international qui définit les règles d'accessibilité web. Niveau AA requis pour les sites publics français (référenciel RGAA).
Bascule entre les méthodes de calcul WCAG2 et APCA
APCA : Accessible Perceptual Contrast Algorithm . Nouvelle méthode de calcul du contraste basée sur la perception humaine. Sera intégrée dans les futures versions de WCAG.
